========================================================
USER STORIES — VIEWER (READ-ONLY STAFF)
Project: American Computers Albania — Supply Chain AI
========================================================

ROLE DESCRIPTION:
Viewers are staff members who need read-only access to the
supply chain system. They can see all data but cannot make
changes or access admin features like user management.

--------------------------------------------------------
STORY 1 — Secure Login
--------------------------------------------------------
As a Viewer,
I want to log in securely with my username and password
So that only authorized staff can access company data.

Acceptance Criteria:
- Login page requires username and password
- Invalid credentials show a clear error message
- Successful login redirects to the main dashboard
- JWT token is stored securely for the session
- Session is automatically invalidated when token expires

--------------------------------------------------------
STORY 2 — Password Reset
--------------------------------------------------------
As a Viewer,
I want to reset my password if I forget it
So that I can regain access to the system without contacting IT.

Acceptance Criteria:
- Forgot password link is visible on the login page
- I receive a password reset email at my registered address
- Reset link takes me to a secure password reset page
- I can set a new password and log in immediately

--------------------------------------------------------
STORY 3 — Dashboard Overview
--------------------------------------------------------
As a Viewer,
I want to see the main supply chain dashboard after logging in
So that I can get a quick overview of company operations.

Acceptance Criteria:
- Dashboard shows key metrics without requiring any action
- Charts load automatically on page visit
- All data is read-only — I cannot modify anything

--------------------------------------------------------
STORY 4 — Navigation Between Sections
--------------------------------------------------------
As a Viewer,
I want to navigate between different sections of the system
So that I can explore inventory, orders, suppliers and AI insights.

Acceptance Criteria:
- Sidebar navigation is always visible
- Clicking a section loads it without page refresh
- Active section is highlighted in the sidebar
- All sections load data from the API automatically

--------------------------------------------------------
STORY 5 — No Access to User Management
--------------------------------------------------------
As a Viewer,
I should not see or access the User Management section
So that sensitive user data is protected from unauthorized viewing.

Acceptance Criteria:
- User Management menu item is hidden for viewer role
- Navigating directly to the users section is not possible
- API returns 403 Forbidden if viewer tries to access user endpoints
